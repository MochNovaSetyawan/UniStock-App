Cara pakai:

  Backup:
  1. Buka XAMPP, pastikan MySQL running
  2. Double-click backup.bat
  3. File ZIP tersimpan di folder htdocs

  Restore di komputer lain:
  1. Install XAMPP
  2. Start Apache + MySQL di XAMPP Control Panel
  3. Extract ZIP ke C:\xampp\htdocs\ (folder apa saja boleh)
  4. Double-click install.bat → browser terbuka otomatis
